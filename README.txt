Purmerend-site — инструкция по развёртыванию на GitHub Pages

Содержимое архива:
- index.html
- style.css
- script.js
- images/ (5 JPG-файлов с горизонтальными фотографиями Purmerend)
- README.txt (этот файл)

Как опубликовать на GitHub Pages:
1. Создайте репозиторий на https://github.com (называйте, например, purmerend-site).
2. В репозитории нажмите "Add file" → "Upload files" и загрузите все файлы из этой папки, включая папку images/.
3. Commit changes.
4. Откройте Settings → Pages (в меню слева) и в разделе Source выберите Branch: main и Folder: / (root). Нажмите Save.
5. Через минуту — несколько минут GitHub подготовит сайт. В секции Pages появится ссылка вида:
   https://ВАШE_ИМЯ.github.io/purmerend-site/
6. Откройте эту ссылку — сайт доступен публично.

Если хотите заменить изображения:
- Поменяйте файлы в папке images/ на свои с теми же именами (purmerend1.jpg ... purmerend5.jpg) или
- В index.html/script.js замените пути на другие.

Примечание насчёт лицензий:
- В данном архиве использованы автоматически сгенерированные изображения-заглушки.
- Если вы хотите использовать реальные фото из интернета, убедитесь, что у вас есть права на их использование или используйте фото с открытой лицензией (например, Wikimedia Commons).

Удачи! Если нужно, помогу загрузить на GitHub шаг за шагом.
